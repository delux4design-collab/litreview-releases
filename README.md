# Scopus Researcher — Official Releases

โปรแกรมช่วยทำ Systematic Literature Review บน Windows
ค้น Scopus → โหลด PDF → AI วิเคราะห์ → จัดกลุ่ม → Export Excel
ทำงานในเครื่องคุณ 100% (รองรับ AI แบบ offline ผ่าน Ollama)

## 📥 ดาวน์โหลด

**[⬇ ดาวน์โหลดเวอร์ชันล่าสุด](https://github.com/delux4design-collab/litreview-releases/releases/latest)** — Windows 10/11 (64-bit), ~45 MB

## สิ่งที่ต้องมีก่อนใช้งาน

- **Scopus API Key** ของตัวเอง — สมัครฟรีที่ <https://dev.elsevier.com>
- เพื่อผลค้นหาเต็มรูปแบบ ควรใช้ในเครือข่ายมหาวิทยาลัย/สถาบันที่บอกรับ Scopus
- พื้นที่ว่าง 5 GB+ / RAM 4 GB+ (แนะนำ GPU NVIDIA — AI เร็วขึ้น 10-50 เท่า)

## ⚠️ Windows SmartScreen

โปรแกรมยังไม่ได้เซ็นใบรับรอง code signing — หาก Windows แสดงจอฟ้า
**"Windows protected your PC"** ให้กด **More info → Run anyway**
